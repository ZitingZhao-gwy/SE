# 账户管理子系统用户手册

本手册面向账户管理系统的工作人员使用者，说明如何登录、如何办理证券账户与资金账户业务、如何查看日志。

本手册讲操作流程，不展开源码实现。

## 1. 使用对象

账户管理系统当前主要面向工作人员使用。

工作人员可以在本系统中办理：

- 证券账户开户、挂失、补办、销户
- 资金账户开户、存款、取款、改密、挂失、补办、销户
- 投资者信息修改
- 操作日志查询

## 2. 登录与认证

### 2.1 工作人员登录

登录页需要输入：

- 工作人员用户名
- 工作人员密码

登录成功后有两种情况：

1. 已经完成过证书认证，直接进入系统主页
2. 首次登录，需要先进入证书认证页

### 2.2 首次登录证书认证

如果系统提示需要证书认证：

1. 进入证书认证页面
2. 输入证书验证码或认证码
3. 提交后完成首次认证
4. 返回系统并建立登录状态

说明：

- 首次证书认证状态由系统保存
- 认证成功后，后续正常使用 token 登录态

## 3. 首页 Dashboard

登录后首先进入 Dashboard。

Dashboard 主要分为三块内容：

### 3.1 统计卡片

会显示：

- 证券账户总数
- 资金账户总数
- 今日新开户数量
- 异常账户提醒数量

说明：

- 当前统计时不再把已销户账户计入总数
- 异常账户一般指冻结状态账户

### 3.2 功能总览

Dashboard 中提供两组快捷业务入口。

证券账户快捷入口：

- 开户
- 挂失
- 补办
- 销户

资金账户快捷入口：

- 开户
- 挂失
- 补办
- 销户
- 改密
- 存款
- 取款

使用方式：

- 点击“开户”会直接进入开户表单
- 点击其他动作会先要求输入账户号，再进入对应校验和操作流程

### 3.3 最新操作记录

Dashboard 中可以查看最近操作日志，并支持条件查询。

当前支持的筛选条件：

- 起始时间
- 结束时间
- 操作类型
- 相关账户号

操作类型当前采用下拉选择。

账户号查询仍然是手动输入，支持：

- `SA...` 证券账户号
- `FA...` 资金账户号

## 4. 证券账户业务

证券账户页面用于管理投资者证券账户。

页面会把账户分为两类：

- 正常账户
- 冻结账户

已销户账户不再作为常规业务列表展示。

## 4.1 证券开户

证券开户分为两类：

- 个人账户
- 法人账户

### 个人账户开户需要填写

- 姓名
- 性别
- 证件类型
- 证件号码
- 联系电话
- 家庭地址
- 工作单位
- 职业
- 学历

如果是代办开户，还需要填写：

- 代办人姓名
- 代办人证件号码

### 法人账户开户需要填写

- 法人名称
- 法定代表人身份证号
- 法人注册登记号
- 营业执照号
- 联系电话
- 联系地址
- 授权执行人姓名
- 授权执行人身份证号
- 授权执行人电话
- 授权执行人地址

### 证券开户注意事项

- 当前系统会校验未成年人不能开户
- 黑名单能力通过外部桥接校验
- 同一投资者不允许重复拥有冲突证券账户

## 4.2 证券挂失

适用场景：

- 证券账户遗失
- 需要冻结证券账户

操作流程：

1. 选择目标证券账户
2. 点击“挂失”
3. 手动输入投资者证件号进行核验
4. 填写挂失原因
5. 提交

效果：

- 证券账户转入冻结状态

## 4.3 证券补办

适用场景：

- 证券账户处于挂失冻结状态后重新补办

操作流程：

1. 在冻结账户列表中找到目标账户
2. 点击“补办”
3. 系统会重新打开开户信息表单
4. 按要求重新填写完整资料
5. 提交补办

效果：

- 原账户关闭
- 新证券账户生成

## 4.4 证券销户

适用场景：

- 投资者申请关闭证券账户

操作流程：

1. 选择正常状态证券账户
2. 点击“销户”
3. 手动输入证件号核验
4. 填写销户原因
5. 提交

重要规则：

- 证券账户必须先清仓
- 如果仍有持仓，系统不允许销户

## 4.5 投资者信息修改

证券账户页面支持修改投资者基础资料。

可修改内容主要包括：

- 姓名
- 性别
- 证件类型
- 证件号码
- 联系电话
- 地址
- 工作单位
- 职业
- 学历

法人相关字段也支持在对应场景下修改。

常见用途：

- 更名
- 联系方式变更
- 地址变更
- 成年后资料更新

## 5. 资金账户业务

资金账户页面用于管理资金账户。

页面会把账户分为两类：

- 正常账户
- 冻结账户

已销户账户不再作为常规列表展示。

## 5.1 资金开户

资金开户前提：

- 已存在证券账户

当前业务规则：

- 先开证券账户
- 后开资金账户
- 开设资金账户时自动与已有证券账户绑定

开户时需要填写：

- 证券账户号
- 投资者证件号
- 币种
- 交易密码
- 取款密码

说明：

- 交易密码与取款密码当前要求为 6 位

## 5.2 存款

操作流程：

1. 在正常资金账户中选择目标账户
2. 点击“存款”
3. 输入金额
4. 输入银行卡号
5. 提交

说明：

- 当前前端提供银行卡号输入框
- 银行卡号暂时只作为界面输入，不提交后端

## 5.3 取款

操作流程：

1. 在正常资金账户中选择目标账户
2. 点击“取款”
3. 输入金额
4. 输入银行卡号
5. 输入取款密码
6. 提交

规则：

- 取款金额必须大于 0
- 取款金额不能超过可用余额
- 需要校验取款密码

## 5.4 修改密码

支持两类密码：

- 交易密码
- 取款密码

操作流程：

1. 在正常资金账户中选择目标账户
2. 点击“改密”
3. 选择密码类型
4. 输入旧密码
5. 输入新密码
6. 提交

规则：

- 旧密码与新密码都要求为 6 位
- 新密码不能与旧密码相同

## 5.5 资金挂失

适用场景：

- 资金卡遗失
- 需要冻结资金账户

操作流程：

1. 选择目标资金账户
2. 点击“挂失”
3. 手动输入投资者证件号
4. 手动输入关联证券账户号
5. 填写挂失原因
6. 提交

当前业务效果：

- 资金账户冻结
- 关联证券账户同步冻结

## 5.6 资金补办

适用场景：

- 资金账户处于冻结状态，需要重新办理

操作流程：

1. 在冻结账户列表中选择目标账户
2. 点击“补办”
3. 输入投资者证件号和证券账户号
4. 设置新的交易密码和取款密码
5. 提交

效果：

- 原资金账户关闭
- 新资金账户生成
- 资金与证券账户关系恢复

## 5.7 资金销户

适用场景：

- 投资者申请关闭资金账户

操作流程：

1. 选择目标资金账户
2. 点击“销户”
3. 输入投资者证件号
4. 填写销户原因
5. 提交

重要规则：

- 如果仍有可用余额或冻结余额，系统不允许销户
- 销户后，证券账户与资金账户分离
- 失去资金账户挂接的证券账户会被冻结

## 5.8 绑定与解绑

当前系统后端已暴露绑定和解绑接口。

用途：

- 资金账户和证券账户建立关系
- 资金账户和证券账户解除关系

前端日常流程中，更多是通过“资金开户自动绑定”和“资金销户自动解绑”来体现。

## 5.9 资金流水查询

在资金账户列表中可对某个账户点击“流水”查看资金流水。

页面会显示：

- 流水时间
- 流水类型
- 金额
- 关联交易号
- 股票名称
- 股票代码
- 股数变化
- 成交价格

说明：

- 当前流水视图不仅包括纯资金流水
- 还会把交易关联的持仓变化信息一起展示

## 6. 日志查询

工作人员操作后，系统会写操作日志。

日志查询入口在 Dashboard。

支持以下条件：

- 按时间范围查
- 按操作类型查
- 按相关证券账户号查
- 按相关资金账户号查

适合用于：

- 复盘某个账户最近做过什么
- 查看某位工作人员最近的业务动作
- 演示系统业务轨迹

## 7. 常见问题

### 7.1 为什么不能直接开资金账户

因为当前业务规则是：

- 必须先开证券账户
- 再开资金账户

### 7.2 为什么证券账户销户失败

常见原因：

- 该证券账户仍有持仓
- 账户状态不是允许销户的状态
- 证件号核验失败

### 7.3 为什么资金账户销户失败

常见原因：

- 仍有可用余额
- 仍有冻结余额
- 账户处于不允许销户的状态

### 7.4 为什么挂失后另一个账户也被冻结了

这是当前业务规则的一部分。

资金账户挂失后，关联证券账户也会联动冻结。

### 7.5 为什么要手动输入证件号

这是工作人员办理敏感账户操作时的人工核验步骤，用于减少误操作。

## 8. 建议使用顺序

对于演示或日常操作，建议按以下顺序使用：

1. 工作人员登录
2. 如果需要，完成首次证书认证
3. 开立证券账户
4. 开立资金账户
5. 办理存款
6. 查看 Dashboard 统计和日志
7. 需要时再演示挂失、补办、销户、改密等动作
