# 界面测试用例方案

## 测试环境

| 项 | 要求 |
|----|------|
| 浏览器 | Chrome 最新版（优先），Edge / Firefox 辅助 |
| 分辨率 | 1920×1080（标准），1366×768（小屏兼容） |
| 后端 | 本地启动 Spring Boot，Mock 数据就绪 |
| Redis | 本地 localhost:6379，含 Mock quote 数据 |
| 测试账号 | 无 Token(GUEST) / 有效 Token(STANDARD) / 有效 Token + is_premium=true(VIP) |

---

## 1. 全局导航栏 (App.vue)

| 编号 | 用例 | 操作步骤 | 预期结果 |
|------|------|----------|----------|
| A1 | 未登录状态导航栏 | 打开首页，不携带 Token | 左侧显示"网上信息发布"logo；中间显示"交易系统""账户系统"外链；右侧显示"登录/注册"按钮 |
| A2 | 已登录 STANDARD | 携带有效 Token 打开首页 | 右侧显示"普通用户"标签（绿色），不显示登录按钮 |
| A3 | 已登录 VIP | 携带 VIP Token 打开首页 | 右侧显示"VIP"标签（橙色） |
| A4 | 子系统跳转 | 点击"交易系统"链接 | 新标签页打开交易系统 URL（当前页不跳转） |
| A5 | 退出登录 | 已登录状态下点击退出 | Token 清除，角色恢复 GUEST，导航栏恢复未登录样式 |

| 编号 | 用例 | 操作步骤 | 预期结果 |

## 5. VIP 升级弹窗 (UpgradeModal.vue)

| 编号 | 用例 | 操作步骤 | 预期结果 |
|------|------|----------|----------|
| U1 | 弹窗展示 | STANDARD 用户点击非 1D 周期按钮 | 背景图表模糊，居中弹窗出现，含锁图标和"升级 VIP"文案 |
| U2 | 模拟支付 | 弹窗中点击"立即升级"按钮 | 调用 `POST /user/upgrade`，成功后弹窗关闭、模糊解除、K 线周期切换生效 |
| U3 | 关闭弹窗 | 点击弹窗外部遮罩或关闭按钮 | 弹窗关闭，模糊解除，K 线恢复 1D 周期 |
| U4 | 已是 VIP | VIP 用户访问任何 K 线周期 | 弹窗永不出现 |
| U5 | 升级失败 | Mock 升级接口返回失败 | 弹窗显示"升级失败，请重试"，不关闭弹窗 |

## 6. 路由与权限

| 编号 | 用例 | 操作步骤 | 预期结果 |

## 7. 异常场景

| 编号 | 用例 | 操作步骤 | 预期结果 |
|------|------|----------|----------|
| E1 | 后端宕机 | 关闭后端，打开前端页面 | 搜索/列表/详情显示"服务繁忙，请稍后重试"或类似降级提示，不白屏 |
| E2 | Redis 不可用 | 关闭 Redis，请求 `/quote` | 后端走 Cache Miss 降级路径，查库回填，或返回错误但不崩溃 |
| E3 | 网络超时 | 浏览器 DevTools 限速 Slow 3G | 请求超时后显示超时提示，不无限 loading |
| E4 | 搜索接口报错 | Mock 搜索接口返回 500 | 列表区域显示错误提示，搜索框仍可继续输入 |
